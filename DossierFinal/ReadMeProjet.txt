Pour utiliser le programme : 
BDD:
Il faut créer une base sous phpMyAdmin avec le nom javaprojet.
Importer le script javaprojet.sql qui se trouve dans le dossier documentation.

Netbeans : 
Il faut ajouter la library mysql lié a netbeans.
Il faut également ajouter les Jar qui se trouve dans le dossier JarExterne.
Dans la classe DbConnexion , il faut vérifier que l'adresse de la BDD est identique ou bien la modifier si nécessaire
